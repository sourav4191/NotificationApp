# NotificationApp

**React Native app with FCM push notifications that work in ALL states (foreground, background, killed)**

## Features

- FCM + Notifee
- Dark/Light theme (system sync + persist)
- OpenWeatherMap API + city search
- Native Kotlin module (battery + storage)
- Redux Toolkit + TypeScript
- React Navigation v6 + navigationRef
- Zero crashes, zero warnings

## Tech Stack
- React Native
- TypeScript
- Redux Toolkit
- React Navigation
- Notifee
- OpenWeatherMap API
- Kotlin (native module)
